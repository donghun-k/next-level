# API Routes 시작

Next.js의 API Routes는 간단하게 서버리스 API를 구축할 수 있는 기능을 제공합니다. 이를 통해 백엔드를 별도로 구축하지 않고도 Next.js 프로젝트에서 데이터 처리를 할 수 있습니다. RESTful API뿐만 아니라 GraphQL API도 쉽게 구현할 수 있습니다.

이번 포스트에서는 간단한 API를 작성하고, 이를 클라이언트에서 호출하는 방법을 단계별로 설명합니다.
